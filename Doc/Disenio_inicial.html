🚦 Diseño de Arquitectura: Semáforos en Moodle

Sistema de Monitoreo de Alumnos - Proyecto Travesía

Introducción del Diseño:
Toda la implementación de los módulos va a estar implementada por la Página de Travesía. Los embebidos en Moodle van a estar diseñados a partir de iframes específicos generados por nuestra página que van a estar divididos de la siguiente manera:
  • Iframe Dashboard Dinámico
  • Iframe Formulario
  • Iframe Configuración Formulario
  • Iframe Ingesta de Datos
  • Iframe Semáforo

Estructura del Sistema: Página Paralela Proyecto Travesía

El núcleo del sistema reside en un servidor propio con una base de datos personal, eliminando dependencias externas de terceros y centralizando la lógica en la plataforma Travesía.

Capas de Procesamiento

  • Almacenamiento: Base de Datos Personal (Local) para persistencia de notas, asistencia y parámetros.
  • Lógica de Control: Módulo de modificación centralizado para ajustar umbrales de alerta y estados del semáforo.
  • Visualización: Generación dinámica de vistas optimizadas para ser servidas como iframe embebido en Moodle.
  • Mensajería: Módulo de notificación que conecta la base de datos con la API de mensajería interna de Moodle.

Tecnologías Core

Base de Datos Personal Página Proyecto Travesía Moodle Web Services HTML5 / iFrames

Seguridad: Validación de tokens entre Moodle y Travesía para asegurar que cada alumno/profesor solo acceda a su información correspondiente.

Detalle de Módulos Embebidos

  • Iframe Dashboard Dinámico: Panel visual de control para el administrador con métricas en tiempo real.
  • Iframe Formulario: Interfaz para la carga de datos específicos por parte de docentes o alumnos.
  • Iframe Configuración Formulario: Herramienta para que el Admin modifique los campos y lógica de los formularios.
  • Iframe Ingesta de Datos: Módulo para la carga masiva de archivos o sincronización con la base de datos personal.
  • Iframe Semáforo: Visualización del estado individual del alumno (Verde/Amarillo/Rojo) basado en los cálculos de Travesía.
